# React интерфейс (Vite + TypeScript)

Фронтенд для Personal Dashboard: авторизация, формы, дашборды,
интеграции и биллинг.

## Запуск локально

```bash
npm install
npm run dev
```

По умолчанию UI доступен на http://localhost:5173  
Backend URL задаётся через `VITE_API_URL`.

## Сборка

```bash
npm run build
```

## Переменные окружения

Создайте `.env` на основе `.env.example`:

```
VITE_API_URL=http://localhost:8000
```
